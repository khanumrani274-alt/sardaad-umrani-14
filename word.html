📄 MS Word 2016 - 15 Kaam ke Shortcuts

Letter, CV, Application typing ke liye speed double kar do.

⌨️ 15 Must-Know Shortcuts

  • CTRL + N = Naya Document
  • CTRL + O = File Open karo
  • CTRL + S = Save karo - har 2 min baad
  • CTRL + P = Print
  • CTRL + B = Bold text
  • CTRL + I = Italic text
  • CTRL + U = Underline
  • CTRL + E = Center me karo
  • CTRL + L = Left align
  • CTRL + R = Right align
  • CTRL + Z = Undo - galti wapas
  • CTRL + Y = Redo
  • CTRL + F = Koi word dhoondo
  • CTRL + H = Replace karo
  • F12 = Save As - naye naam se save

📝 CV/Letter Tip

1. Page Layout > Margins > Narrow karo, zyada content aayega.
2. Insert > Page Number se page no. lagao.
3. Save karte waqt .docx ki jagah .pdf bhi select kar sakte ho.
