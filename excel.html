🚀 10 Pro Formulas/Tricks

  • VLOOKUP = Dusri sheet se data uthao: =VLOOKUP(A2, Sheet2!A:B, 2, 0)
  • IFERROR = #N/A error chupao: =IFERROR(VLOOKUP(...), "Nahi Mila")
  • Conditional Formatting = Home > Conditional Formatting > Salary >50000 ko green karo
  • Freeze Panes = View > Freeze Panes. Heading upar fix ho jayegi
  • Remove Duplicates = Data > Remove Duplicates. Ek hi naam 2 baar nahi aayega
  • Pivot Table = Insert > PivotTable
